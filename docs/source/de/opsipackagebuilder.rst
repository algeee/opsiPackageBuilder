opsi PackageBuilder Handbuch
============================

|image0|\  opsi Package Builder

opsi Package Builder ist ein Werkzeug zur Erstellung und Verwaltung von Softwarepaketen in Zusammenarbeit mit dem opsi (open pc server integration) Clientmanagementsystem der Firma uib GmbH.

Hauptmerkmale der Anwendung sind:

-  Paketstrukturen anlegen und wiederholt bearbeiten
-  Abhängigkeiten und Produktvariablen anlegen, löschen, bearbeiten
-  Changelog pflegen
-  Skriptverwaltung, inkl. eigenem SkriptEditor
-  Paketrechte setzen
-  Pakete bauen, installieren und entfernen
-  Zeitplaner: zeitgesteuerte Installationsaufträge per AT Job am Server verwalten (inkl. on\_demand und WakeOnLan)
