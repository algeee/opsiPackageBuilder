Einfacher Editor
================

|image27|

Ist in den :ref:`Programmeinstellungen <settings_program>` die Nutzung des erweiterten Changelog Editors deaktiviert, erscheint beim Klick auf die Schaltfläche "Changelog" im :ref:`tabpacket` das einfache Editorfenster. Hier kann reiner Fließtext in beliebiger Form als Hinterlegt werden.

Wird der Editor über die Schaltfläche \ |image28|\  geschlossen, werden autom. sämtliche Änderungen übernommen.
